# Superstore Sales Data Analysis

## Project Overview
This project analyzes the Superstore Sales dataset to understand sales performance, profit trends, and customer segments. The goal is to extract useful insights that help improve business decisions.

## Dataset
The dataset contains sales records from a retail superstore, including information about:
- Orders
- Customers
- Products
- Sales
- Profit
- Regions and Segments

## Tools Used
- Microsoft Excel
- Data Cleaning
- Data Analysis
- Pivot Tables
- Data Visualization

## Key Analysis
The analysis focuses on:

- Total Sales and Profit
- Sales by Category and Sub-Category
- Sales by Region
- Customer Segments
- Profitability trends

## Key Insights
- Identifying the most profitable product categories
- Understanding which regions generate the highest sales
- Analyzing customer segments and their contribution to revenue

## Project Goal
The goal of this project is to demonstrate data analysis skills and the ability to turn raw data into meaningful insights.
